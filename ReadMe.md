### Teams

- **Create Team:**
  `POST /api/teams`
- **Get All Teams:**
  `GET /api/teams`
- **Get Team by ID:**
  `GET /api/teams/:teamId`
- **Update Team:**
  `PUT /api/teams/:teamId`
- **Delete Team:**
  `DELETE /api/teams/:teamId`
